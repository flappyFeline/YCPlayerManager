## Installation

### Manual  
Drag the Classes directory to your project directly.

### Cocoapods
YCPlayerManager is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod "YCPlayerManager"
```
